# 📊 Informe PRO: La Estrategia de Francisco Doglio

Este documento resume el sistema de crecimiento y monetización de @franciscodoglio aplicado a una agencia de IA.

## 💎 Filosofía: Autoridad sobre Engagement
Francisco no busca ser "viral" por fama, sino por negocio. Cada video es un activo que filtra clientes.

## 🛠️ El Doglio Framework para Reels
1. **Hook (0-3s):** Atacar el bolsillo o el estatus con datos reales.
2. **Cuerpo (4-30s):** Explicar el "Cómo" de forma rápida usando una pizarra o esquemas visuales.
3. **Prueba:** Mostrar capturas de pantalla de resultados durante el video.
4. **CTA:** Palabra clave directa (ej: "COMUNIDAD", "SISTEMA").

## ⚙️ Configuración Técnica Recomendada
- **Edición:** Ritmo rápido, sin silencios, textos grandes y dinámicos.
- **Ads:** $5-$10 diarios en los Reels que ya tienen mejor alcance orgánico.
- **Automatización:** n8n debe interceptar el comentario y enviar el Lead Magnet al instante.

## 🎯 Plan Maestro para Automatizame.uy
- **Look Seba CEO:** Mantener consistencia visual (branding).
- **Contenido:** Mostrar el "detrás de escena" de los workflows de n8n funcionando en tiempo real.
