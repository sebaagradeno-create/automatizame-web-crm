# 🤖 Revolutia.ai: La Revolución de los Agentes con Rowboat

Investigación profunda sobre los sistemas de agentes y automatización presentados por @revolutia.ai.

## 🚣 Rowboat: El Constructor de Agentes YC
Plataforma de código abierto para desplegar enjambres de agentes con memoria infinita.
- **Gráfico de Conocimiento:** Conecta con tu ecosistema de Google para que la IA sepa todo sobre tu negocio.
- **Despliegue Rápido:** De idea a agente funcional en segundos vía lenguaje natural.
- **Privacidad:** Compatible con Ollama para procesar datos localmente.

## 🛠️ Herramientas y Patrones Clave
- **VibeProspecting:** Sistema de scraping y prospección integrado en Claude.
- **YCloud + n8n:** La combinación ganadora para WhatsApp empresarial oficial.
- **AirLLM:** Optimización de infraestructura para correr modelos grandes en PC antiguas.
- **Skills.md:** Técnica para estandarizar la calidad de las respuestas de los agentes de la agencia.

## 🎯 Plan Maestro para Automatizame.uy
- Utilizar **Rowboat** como nuestra herramienta de "Prototipado Rápido" para clientes.
- Implementar **VibeProspecting** para captar clientes para nuestra propia agencia de forma automática.
