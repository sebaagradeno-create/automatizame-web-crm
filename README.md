# 🤖 Agencia Automatizame - Coworking IA

Este es el repositorio central de la **Agencia Automatizame**. Está diseñado para el trabajo colaborativo entre agentes de IA (Claude y Antigravity) y el equipo humano.

## 📂 Estructura del Proyecto

- **`.agentes/`**: 🧠 Configuración, prompts y reglas específicas para cada IA.
  - `claude/`: Instrucciones para Claude Code / Desktop App.
  - `antigravity/`: Instrucciones para Antigravity (Google DeepMind).
- **`assets/`**: 🖼️ Recursos visuales, logos, iconos y capturas de pantalla.
- **`docs/`**: 📚 Documentación técnica, esquemas de DB, credenciales y estados de agentes.
- **`marketing/`**: 📣 Activos de marca y estrategias de lanzamiento.
- **`n8n_workflows/`**: ⚙️ Automatizaciones de n8n exportadas en JSON.
- **`scripts/`**: 💻 Scripts de PowerShell y herramientas de automatización local.
- **`src/`**: 🏗️ Código fuente de la web y aplicaciones.

## 🛠️ Reglas de Oro de Coworking

1. **Estructura Primero**: Antes de crear un nuevo módulo, se debe proponer la estructura de archivos en la carpeta correspondiente.
2. **Documentar Cambios**: Al finalizar una tarea, actualizar el archivo de estado correspondiente en `docs/`.
3. **Comunicación en Español**: Todas las interacciones deben ser en español claro y directo.
4. **Respeto de Espacios**: No modificar archivos dentro de `.agentes/` de la otra IA sin previo aviso.
